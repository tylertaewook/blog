Being a heavy Sci-fi fan myself, I always wondered: how would those spherical wheels from *Tron* and *I-Robot* work in real life?
And this simple thought began the 6-month journey of **Project Orbitron**.

Now this project consisted of two major goals upon start:
* Building a vehicle with spherical wheels that implement a 4 wheel independent steering/driving (4WIS/D) system using Arduino
* Developing a intuitive control algorithm for 4WIS/D vehicle in Mathematica

In this article, I will showcase my vehicle prototype Orbitron along with a short story behind the building scene. Then, I'll introduce you to the clever algorithm I built that is single-handedly based on a very simple physics concept.

## Orbitron

As I mentioned above, ORBITRON is a vehicle with spherical wheels, hence the name 'ORB'itron. Unfortunately I was *a bit* under-qualified to suspend wheels in mid-air with electromagnets, as many sci-fi movies suggest. So I implemented a 4 Wheel Independent Steering/Driving (4WIS/D) system which is a fancy word for saying that each wheel can rotate independently.

![preview](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/njfaaykq0a4e6md4yr20.png)

Although I received funds from Kent School Pre-Engineering Center, this prototype ended up costing $1,000+ and 60+ work hours to build.

### Wheels



![designwheel](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/me49fpx81wvg4gxgd82z.jpg)


![wheel](https://dev-to-uploads.s3.amazonaws.com/uploads/articles/lvok6be2jiqc0jwtcm6x.png)

Simply put, Orbitron is a vehicle with spherical wheels. The spherical wheels allow the efficient implementation of a 4WIS/4WID (4 Wheel Independent Steering/Driving) system. 4WIS/4WID is a steering system for a four-wheeled vehicle that allows for separate speed and direction controls for each wheel.
To accomplish this, each wheel has two separate motors each controlling speed and direction.



While building the Orbitron, a simple C# WinForms application was developed. This application communicates with Orbitron via XBee wireless module and controls simple movements to ensure every component is working properly.
